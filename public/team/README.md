# Fotos da Equipe

## Como adicionar fotos

1. Coloque a foto do CEO em: `public/team/ceo.jpeg`
2. Para adicionar mais membros, coloque as fotos aqui seguindo o padrão: `public/team/[nome-arquivo].jpeg` ou `.jpg`

## Foto do CEO

- Arquivo esperado: `ceo.jpeg` ou `CEO.jpeg`
- Localização: Esta pasta (`public/team/`)
- Se a foto estiver na raiz do projeto, mova-a para esta pasta e renomeie para `ceo.jpeg`
